 NOSQL 是什么
咱们先来列举一下传统型数据库的特点：

结构化
二维表
E-R 关系（实体 - 关系模型）
sql 标准化
支持事务（ACID）
锁
索引
sql ，是结构化查询语言，泛指关系型数据库

nosql （not noly sql），不仅仅是 sql ，这泛指不提供 sql 功能的非关系型数据库

它不遵循 sql 的标准，acid 特性，表结构等特性。

最开始 nosql 实际上是 not sql ，后面慢慢发展成 not only sql
