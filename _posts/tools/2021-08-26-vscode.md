```json
      "args": ["toutiao"] // 参数
    }
  ]
}
```

F5 调试运行

### Debug actions
Once a debug session starts, the Debug toolbar will appear on the top of the editor.

Debug Actions

Continue / Pause F5
Step Over F10
Step Into F11
Step Out ⇧F11
Restart ⇧⌘F5
Stop ⇧F5
